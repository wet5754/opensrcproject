20년도 오픈소스 소프트웨어 입문 파이널 프로젝트.

본 프로젝트는 네이버 API에서 받은 데이터를 그래프로 시각화 하는 프로젝트.

- 출력 기간
19년 10월 1일 ~ 20년 5월 31일

- 파일 구성

opensrcproject : 본 프로젝트가 들어있는 파일
opensrcproject_test : 프로젝트 진행 중 parser와 naverapi부분의 디버깅을 위한 테스트 프로젝트

해당 부분을 test로 사용함 */

사용된 오픈소스 : maven, tomcat, google chart, json-simple 등.



/* 주의!!! - 그래프 출력에서의 제한사항. */

naverAPI에서 제공하는 검색량 트랜드 JSON파일이 "모든 날짜"에 대한 정보를 담지 않음.

따라서 추세선 중간에 공백이 존재함.

이러한 현상은 키워드의 숫자가 2개 이하일 때 심해진다.(3개 이상의 키워드 권장)


<해결 방향>
NaverAPI에서 받은 JSON에서 공백의 갯수를 구함.

데이터가 존재하는 구간 중 "랜덤한 위치를 공백의 갯수"만큼 구한 뒤, 동일한 데이터 값을 중복 삽입

정확한 날짜값은 변동되나, 전체적인 검색량 트렌드를 유지
